### Dependencies:
- sbcl
- quicklisp
  - str
  - alexandria

### Run day
```bash
$ sbcl --script ./main.lisp <day>
# for example...
$ sbcl --script ./main.lisp 1
# or...
$ ./main.lisp 1
```

### Run all
```bash
$ sbcl --script ./main.lisp all
# or...
$ ./main.lisp all
```
